Step 1: Telegram API Credentials

Get your API ID and Hash from Telegram

  1. 1

    Visit Telegram's Developer Portal

    Go to my.telegram.org and log in with your phone number

  2. 2

    Navigate to API Development Tools

    Click on "API Development Tools" in the menu

  3. 3

    Create a New Application

    Fill in the form with any app name and short name. You'll receive your API ID and API Hash

⚠️
Keep these credentials safe! Never share your API Hash publicly. Store them in your .env file.
📊

Step 2: Google Service Account

Setup Google Sheets API access

  1. 1

    Go to Google Cloud Console

    Visit console.cloud.google.com

  2. 2

    Create or Select a Project

    Create a new project or use an existing one

  3. 3

    Enable Google Sheets API

    Go to APIs & Services → Enable APIs → Search for "Google Sheets API" → Enable

  4. 4

    Create Service Account

    Go to IAM & Admin → Service Accounts → Create Service Account → Download JSON key

  5. 5

    Create & Share Google Sheet

    Create a blank Google Sheet and share it with the service account email (found in the JSON file)

  6. 6

    Copy Sheet ID

    From the URL: docs.google.com/spreadsheets/d/SHEET_ID/edit

🔐

Step 3: Local Authentication

One-time Telegram session setup

# Install dependencies pip install -r requirements.txt # Copy environment template cp .env.example .env # Edit .env with your credentials # TG_API_ID, TG_API_HASH, TG_PHONE, GOOGLE_CREDS_JSON, SHEET_ID # Authenticate with Telegram (enter the code they send you) python auth_session.py # Base64 encode the session for GitHub python -c "import base64; print(base64.b64encode(open('scraper_session.session','rb').read()).decode())"
💡
Tip When running auth_session.py, Telegram will send a code to your phone. Enter it when prompted.
🐙

Step 4: GitHub Repository Setup

Configure secrets for automated scraping

  1. 1

    Create a New GitHub Repository

    Create a new repo and push this project to it

  2. 2

    Add Repository Secrets

    Go to Settings → Secrets and variables → Actions → New repository secret

Secret Name Value
TG_API_ID Your Telegram API ID
TG_API_HASH Your Telegram API Hash
TG_PHONE Your phone number (+...)
TG_SESSION Base64-encoded session file
GOOGLE_CREDS Base64-encoded service-account.json
SHEET_ID Your Google Sheet ID
# Base64 encode Google credentials python -c "import base64; print(base64.b64encode(open('service-account.json','rb').read()).decode())"
📋

Step 5: Configure Channels

Edit channels.json to add your target channels

{ "default_hours": 24, "channels": [ {"name": "DarbMirror", "hours_back": 24}, {"name": "AnotherChannel", "hours_back": 48} ] }
📝
Channel Configuration
  • name: Channel username (without @)
  • hours_back: How many hours of messages to fetch
🚀

Step 6: Run the Scraper

Execute locally or let GitHub Actions handle it

Manual (Local)

python main.py

Automated (GitHub Actions)

  • ⏰

    Automatic Schedule

    Runs daily at 06:00 UTC (09:00 UTC+3) automatically

  • ▶️

    Manual Trigger

    GitHub repo → Actions tab → "Daily Telegram Scraper" → Run workflow

✅
You're all set! Once configured, the scraper will automatically fetch messages and populate your Google Sheet daily.
