Scheduler

Automate scraping with scheduled jobs

The global scheduler runs the Dashboard quick scrape at a set time or interval. Each user also has per-process scheduling inside Profiles — every process can have its own daily time or hourly interval, running independently. This means each user effectively has their own cron jobs without sharing the admin's schedule.
